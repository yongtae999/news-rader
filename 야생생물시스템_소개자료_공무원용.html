Core Features

앱의 핵심 기능 3가지

수렵인이 현장에서 경험하는 가장 강력한 변화

🛡️

① 안전레이더

활동 시작 시 반경 200m 내 동료의 스마트폰 위치를 실시간 감지하여 오인 사격 등 인명 사고를 원천 예방합니다.

📷

② 간편 포획 등록

복잡한 서류 없이 동물 아이콘 터치 → 현장 사진 촬영 → 세부 정보 입력의 3단계로 포획 신고를 즉시 완료합니다.

💰

③ 신속 포상금 처리

앱에 저장된 계좌 정보와 포획 데이터가 지자체 담당자에게 즉시 전송되어 행정 절차 및 포상금 지급이 빨라집니다.

💡 GPS 자동 기록: 포획 등록 시 스마트폰 GPS가 현재 주소와 위·경도 좌표를 자동으로 입력하여 허위 신고 원천 차단

지자체 관리자 웹 시스템

포획 현황 관리부터 실시간 안전 관제까지, 하나의 웹으로 모든 업무 처리

📊

대시보드 — 통합 현황 한 눈에

실시간 총 포획 건수, 승인 대기 건수, 활성 사용자 수 등 핵심 KPI와 월별 포획 추이 그래프를 첫 화면에서 즉시 확인

🗺️

실시간 관제 — 현장 인력 위치 모니터링

활동 중인 모든 수렵인의 현재 위치를 지도 위에 실시간 표시. 안전 상태(정상/비정상) 및 디바이스 연결 여부 즉각 파악 가능

✅

포획 심사 — 사진·GPS 확인 후 승인/반려

엽사 신고 현장 사진과 GPS 좌표, 위치이력 동선을 교차 검증한 뒤 승인 또는 반려 처리. 지급조서를 엑셀로 즉시 내보내기 가능

👥

사용자 관리 — 엽사 계정 등록 및 권한 설정

이름·연락처·역할·소속 지부 등록. 등록된 번호로만 앱 로그인 가능하여 미인가 인원의 무단 사용을 원천 차단

Q&A

자주 묻는 질문

도입을 검토 중인 지자체 담당자가 주로 묻는 질문

특별한 장비나 별도 SW 구매가 필요한가요?
아닙니다. 수렵인은 본인 스마트폰(안드로이드/iOS)에 앱만 설치하면 되고, 지자체 담당자는 일반 PC 웹 브라우저로 바로 접속합니다. 별도 장비 구매 비용이 없습니다.
산속에서 통신이 안 되는 음영 지역은 어떻게 하나요?
음영 지역에서는 포획 데이터가 앱에 임시 저장됩니다. 통신이 가능한 지역으로 이동하면 '미등록 탭'에서 저장된 데이터를 확인하고 서버로 재전송할 수 있습니다.
기존에 Excel로 관리하던 데이터와 연동이 가능한가요?
포획 데이터를 엑셀 형태의 지급조서로 내보낼 수 있어 기존 행정 시스템과 연계 활용이 가능합니다. 기간·종별·지역별 필터 설정 후 원하는 범위의 데이터만 추출할 수 있습니다.
